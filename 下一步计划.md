## 共用约定

### 创意背景

- **设置入口**：设置 → 外观 → 创意背景（效果卡片 + 强度 10–200%）
- **存储**：`localStorage` → `toolbox-appearance` → `creativeBackground`
- **禁用叠加**：`colorScheme === 'custom'` 时不渲染创意背景 Canvas（与现逻辑一致）
- **组件**：`app/src/components/layout/effects/RainRipplesEffect.vue`

### 自定义皮肤 · 预设视频

- **设置入口**：设置 → 外观 → 视频皮肤预设
- **资源目录**：`workspaces/skin-presets/`

## 验收标准（雨滴涟漪）

1. 设置 → 创意背景 → 选「雨滴涟漪」后，主界面可见**动态涟漪**（非视频背景）
2. 强度滑块影响落点频率与可见度；选「无」后立即消失
3. `colorScheme === 'custom'` 时不与创意背景叠用
4. 顶栏/卡片可点；`npm run build` 通过
